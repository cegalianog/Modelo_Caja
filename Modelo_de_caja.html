Modelo de caja

Teoría de las Dimensiones de la caja

El concepto de modelo de caja (Box Model), se conoce como contenedor o cajas rectangulares que se encuentran dentro de un documento HTML; cada contenedor puede tener dentro de ella textos, imágenes, links y botones, etc.

Para empezar a explicar sus propiedades de modelo de caja se debe tener en cuenta su ancho y alto (width y height), por la cual hace semejanza a las 2 propiedades principales de del modelo de caja que define el ancho y alto de nuestra caja.

Para esto explicaremos en detalle como CSS nos muestra sus variantes de ancho y alto:

Propiedad Valor Significado
width: auto | [tamaño] Tamaño de ancho de un elemento.
max-width: none | [tamaño] Ancho máximo que puede ocupar un elemento.
min-width: 0 | [tamaño] Ancho mínimo que puede ocupar un elemento.
height: auto | [tamaño] Tamaño de alto de un elemento.
max-height: none | [tamaño] Alto máximo que puede ocupar un elemento.
min-height: 0 | [tamaño] Alto mínimo que puede ocupar un elemento.

Como podemos ver las anteriores propiedades que nacen del ancho y alto, hacen que tengamos más flexibilidad a la hora de hacer un ancho y un alto para un elemento. Ahora, “max-“ y “min-“, son propiedades que determinan los anchos y altos máximos y mínimos de este mismo. Cabe aclarar que “min-width” y “min-height”, por defecto tienen valor de 0, y “max-width” y “max-height”, su valor por defecto “none”.

Una vez comprendido esto, es importante mencionar sus dimensiones de la caja, como lo son Margin, Border y Padding, con sus segmentos left, right, top y bottom.

Límite de Contenido

Esto define el perímetro de cada una de las cuatro áreas (contenido, padding, border y margin) es llamado "límite", de manera que cada caja tiene cuatro límites:

  • límite del contenido o límite interno: El límite del contenido rodea al contenido procesado del elemento.
  • límite de Padding: El límite de padding (relleno) rodea a la caja de relleno. Si padding tiene un ancho de 0, el límite del relleno es el mismo que el límite de contenido. El límite del relleno de una caja define el límite de la caja de contención establecida por la caja.
  • límite de border: El límite de border (borde) rodea el borde de la caja. Si border tiene un ancho de 0, el límite del borde es el mismo que el límite del relleno (padding).
  • límite de margin o límite externo: El límite de margin (margen) rodea el margen de la caja. Si margin tiene un ancho de 0, el límite del margen es el mismo que el límite del borde (border).

Veamos un ejemplo:

Para cada límite puede dividirse en:

  • Top: Se refiere a la parte superior del elemento.
  • Left: Se refiere a la parte izquierda del elemento.
  • Right: Se refiere a la parte derecha del elemento.
  • Bottom: Se refiere a la parte inferior del elemento.

Un ejemplo de cómo se escribe es:

.clase{
    margin-left : …;
    padding-left : …;
    width : …;
    padding-right : …;
    margin-right : …;
    margin-top : …;
    padding-bottom : …;
}

Cada caja de modelo de caja se les puede añadir una propiedad llamada “border”, que se refiere al contorno del elemento.

Las propiedades de los bordes en CSS son las siguientes:

Propiedad Valor Significado
border-color: [color] Especifica el color que se utilizará en el borde.
border-width: thin | medium | thick Especifica un tamaño predefinido para el grosor del borde.
[tamaño] También se puede indicar el tamaño concreto del grosor del borde.
border-style: none No utiliza ningún estilo para el borde (igual a grosor de borde 0).
hidden Igual a 'none', excepto en términos de resolución de conflictos de bordes para los elementos de tabla.
dotted El borde es una serie de puntos.
dashed El borde es una serie de pequeños segmentos de línea.
solid Establece un borde sólido (línea continua).
double Establece un borde doble (dos líneas continuas).
groove Establece un borde biselado con luz desde arriba.
ridge Establece un borde biselado con luz desde abajo. Opuesto a groove.
inset Establece un borde con profundidad «hacia dentro».
outset Establece un borde con profundidad «hacia fuera». Opuesto a inset.
